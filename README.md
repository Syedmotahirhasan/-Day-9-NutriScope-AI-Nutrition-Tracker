# Day 9 – NutriScope: AI Nutrition Tracker

## Task
Build a nutrition tracking app (NutriScope) using Claude AI – MVP version and Enhanced version.

## Tools Used
- Claude AI (claude.ai)
- HTML, CSS, JavaScript

---

## MVP Version
A basic nutrition tracker with:
- Food search from offline database
- Add food to daily log
- View total calories, protein, carbs, fat

---

## Enhanced Version
An improved version with:
- Meal categories (Breakfast, Lunch, Dinner, Snack)
- Daily goals with progress bars
- Macro donut chart visualization
- AI-powered nutrition analysis using Claude API
- Better UI with cards, tabs, and animations

---

## Key Learnings
1. Prompting Claude with clear requirements generates working code fast.
2. MVP-first approach helps understand what features to add next.
3. AI can be integrated directly into web apps via Anthropic API.
4. Iterative building (MVP → Enhanced) is a practical real-world skill.
5. Claude understands Indian food items (dal, roti, paneer) which makes it locally relevant.
